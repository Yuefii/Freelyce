<p align="center">
  Freelyce adalah aplikasi invoice generator berbasis web yang dirancang untuk para freelancer, pekerja mandiri, dan pelaku bisnis kecil. Aplikasi ini berfokus pada kesederhanaan, kecepatan, dan profesionalitas, membantu Anda membuat invoice yang rapi dan terpercaya dalam hitungan menit.
</p>

---

### Fitur Utama

- **Pembuatan Invoice Cepat**: Buat invoice profesional dalam format PDF dengan cepat.
- **Nomor Invoice Otomatis**: Hasilkan nomor invoice unik untuk setiap transaksi.
- **Perhitungan Fleksibel**: Mendukung perhitungan subtotal, diskon (persentase atau nominal tetap), dan pajak.
- **Kustomisasi Mudah**: Tambahkan item, catatan, dan detail penting lainnya dengan mudah.
- **Antarmuka Web Sederhana**: Dilengkapi dengan antarmuka statis untuk interaksi langsung.
