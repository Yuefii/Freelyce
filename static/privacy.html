Kebijakan Privasi

Terakhir diperbarui: 18 November 2025

Selamat datang di Freelyce. Kami menghargai privasi Anda dan berkomitmen untuk melindunginya. Kebijakan Privasi ini menjelaskan bagaimana kami menangani informasi Anda saat menggunakan layanan kami.

1. Tidak Ada Pengumpulan Data Pribadi

Freelyce dirancang dengan prinsip privasi sebagai prioritas utama. Kami tidak memerlukan registrasi akun untuk menggunakan layanan kami. Semua fitur inti, seperti pembuatan dan pengunduhan invoice, dapat diakses secara anonim.

Sebagai project open source, kami percaya pada transparansi. Anda dapat memeriksa kode sumber kami kapan saja untuk memverifikasi bahwa kami mematuhi klaim privasi kami. Kunjungi repositori kami di Repositori GitHub Freelyce.

2. Informasi yang Anda Masukkan

Saat Anda membuat invoice, Anda memasukkan data seperti nama, alamat, detail item, dan informasi kontak. Informasi ini hanya digunakan untuk tujuan berikut:

  • Membuat Dokumen PDF: Data yang Anda berikan diproses di server kami untuk menghasilkan file invoice dalam format PDF.
  • Transmisi Sementara: Informasi tersebut dikirimkan secara aman ke server kami untuk diproses dan segera dihapus setelah file PDF dibuat dan dikirimkan kepada Anda.

Kami tidak menyimpan data invoice Anda di server kami. Setelah proses pembuatan PDF selesai, semua informasi yang terkait dengan invoice tersebut akan dihapus secara permanen dari sistem kami.

3. Riwayat Invoice (Penyimpanan Lokal)

Freelyce menyediakan fitur "Riwayat" yang memungkinkan Anda melihat invoice yang pernah dibuat sebelumnya. Fitur ini bekerja dengan menyimpan data invoice langsung di browser Anda (menggunakan Local Storage), bukan di server kami. Ini berarti:

  • Hanya Anda yang dapat mengakses riwayat invoice Anda dari perangkat dan browser yang sama.
  • Data riwayat tidak pernah dikirim atau disimpan di server Freelyce.
  • Jika Anda membersihkan cache browser atau menggunakan perangkat lain, riwayat tersebut tidak akan tersedia.

4. Logo Perusahaan/Bisnis

Jika Anda memilih untuk mengunggah logo, file gambar tersebut akan diproses untuk disertakan dalam invoice PDF. Sama seperti data lainnya, file logo tidak disimpan di server kami dan akan dihapus setelah proses pembuatan PDF selesai.

5. Data Analitik Anonim

Kami mungkin mengumpulkan data penggunaan anonim (seperti jumlah kunjungan atau fitur yang paling sering digunakan) untuk tujuan meningkatkan kualitas layanan kami. Data ini bersifat agregat dan tidak dapat dihubungkan dengan individu mana pun.

6. Perubahan pada Kebijakan Ini

Kami dapat memperbarui Kebijakan Privasi ini dari waktu ke waktu. Perubahan akan dipublikasikan di halaman ini, dan kami menyarankan Anda untuk meninjaunya secara berkala.

Hubungi Kami

Jika Anda memiliki pertanyaan lebih lanjut mengenai kebijakan privasi kami, jangan ragu untuk membuka isu (issue) di Repositori GitHub Freelyce.
