Dokumentasi API

Kami juga menyediakan API untuk para pengembang. Berikut adalah panduan untuk menggunakan API untuk membuat invoice.

GET /next-invoice-number

Mendapatkan nomor invoice berikutnya yang tersedia dalam format sekuensial.

Method

GET

Endpoint

https://freelyce.vercel.app/next-invoice-number

Parameters

Tidak ada.

Success Response (200 OK)

Mengembalikan objek JSON dengan nomor invoice berikutnya.

{"invoice_number": "INV-2025-0001"}

POST /invoice

Membuat file invoice dalam format PDF berdasarkan data JSON yang dikirim.

Method

POST

Endpoint

https://freelyce.vercel.app/invoice

Request Body

Anda harus mengirimkan objek JSON dengan struktur berikut di dalam body request:

Invoice Object Fields:

Field Type Required Description
invoice_number string Ya Nomor unik untuk invoice.
from_address string Ya Alamat pengirim invoice.
bill_to_address string Ya Alamat penerima invoice.
date string (DD-MM-YYYY) Ya Tanggal pembuatan invoice.
due_date string (DD-MM-YYYY) Ya Tanggal jatuh tempo pembayaran.
po_number string Tidak Nomor Purchase Order (opsional).
discount float Tidak (default: 0.0) Jumlah diskon yang diterapkan.
discount_type string Tidak (default: "fixed") Tipe diskon ("fixed" atau "percentage").
items array of Item objects Ya Daftar item atau layanan dalam invoice.
notes string Tidak Catatan tambahan untuk invoice.
tax_rate float Tidak (default: 0.0) Tingkat pajak dalam persentase.
logo string Tidak Gambar logo dalam format base64 / url logo (opsional).

Item Object Fields (nested in 'items' array):

Field Type Required Description
description string Ya Deskripsi item atau layanan.
quantity integer Ya Jumlah item.
rate float Ya Harga per unit item.

Berikut adalah contoh lengkap dari body request:

{
    "invoice_number": "INV-2025-0001",
    "from_address": "Freelyce\nJl. Merdeka No. 10\nJakarta, Indonesia",
    "bill_to_address": "Bapak John\nJl. Sudirman No. 5\nSurabaya, Indonesia",
    "date": "2025-11-18",
    "due_date": "2025-12-18",
    "po_number": "PO-12345",
    "discount": 5,
    "discount_type": "percentage",
    "items": [
        {
            "description": "Jasa Desain Logo",
            "quantity": 1,
            "rate": 500000.0
        },
        {
            "description": "Lisensi Software",
            "quantity": 2,
            "rate": 150000.0
        }
    ],
    "notes": "Tolong lakukan pembayaran sebelum tanggal jatuh tempo. Terimakasih telah menggunakan jasa kami.",
    "tax_rate": 0,
    "logo": "https://freelyce.vercel.app/static/img/logo-white-background.png"
}

Success Response (200 OK)

Mengembalikan file biner dengan `Content-Type: application/pdf`.

Error Response (422 Unprocessable Entity)

Terjadi jika body request tidak sesuai dengan skema yang dibutuhkan. Respon akan berisi detail error validasi.
